#自带过滤器和标签

  • 模板语法自己查，反正记不住
  • 小写过滤器 ('Name'|lower) => {{ 'Name'|lower }}
  • 嵌套使用 ('Name'|first|upper) => {{ 'Name'|first|upper }}
  • 切取后面...显示('a b c d e f g'|truncatewords:"6") => {{'a b c d e f g'|truncatewords:"6"}}
  • date过滤器 {{ date|date:"Y-m-d H:m:s" }}
  • safe解析html标签 {{ label|safe }}
  • 外部引入include {% include "a.html" %}
  • (name|default:"设置默认值") => {{ name|default:"设置默认值" }}
  • ('Hello Django'|length) => {{ name|length }}
  • (1024|filesizeformat) => {{ num|filesizeformat }}
  • csrf_token 用于form表单中，作用是跨站请求伪造保护。如果不用｛% csrf_token %｝标签，在用 form 表单时，要再次跳转页面会报403权限错误。用了｛% csrf_token %｝标签，在 form 表单提交数据时，才会成功。

#自定义过滤器和标签

1 新建templatetags

2 my_tags.py

3 settings.py里TEMPLATES-libraries 'my_tags':'templatetags.my_tags'

4 引入load my_tags

@register.filter 只能传递两个变量，多个的话要用循环 能在%if变量判断时使用

@register.simple)_tag 能传递多个变量 不能在%if变量判断时使用

  • 自定义过滤器: {{ 11|my_filter:22 }}
  • 自定义标签: {% my_tag1 11 22 33 %}
